## 방법 

1. 매주 동일 논문 읽고 발제문 제출
2. 한 명씩 돌아가면서 발표
3. 발표한 영상 유튜브 업로드
  - 가능한 한 발표자가 영상 만들기 (zoom으로 방장이 녹화 시 연결 불량 문제로 끊김 부분이 함께 녹화되는 경우가 있다) 
4. 자신의 발표 차례가 아니어도 발표 영상을 만드신 분은 언제든 공유 환영.
